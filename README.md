# vcmp-archive
Hello,

This is a collection of old VC-MP versions starting from 2005, I decided to save it as
a simple google search doesn't find you versions older than 0.3z r2 and there are actually a lot of vc-mp versions
each made by different people, with different playstyle and features for example there are older versions with features that the current vc-mp doesnt have and we used to host 0.3 server and used to play regularly as we found it very fun. they might be really worth checking it out if you like vcmp.

I will add notes here if needed;

* Note 1: Past 0.4 versions are lost media for example 04rel002 with custom splash screen is nowhere to be found.
* Note 2: While i had more 0.4 stuff i decided not to archive them for the following reasons; \
1-) 0.4 client and servers are very complex to archive, there are lot of server and client files for one version, i.e. 04rel002 has like 5 different builds
and there is no way to understand which one is compatible with which server build. \
2-) Even if i upload an old 0.4 client and server it would not be useful since 0.4 servers do not come with plugins by default and
those said plugins are not archived properly, so it's not possible to find plugin packages compatible with older versions,
Without plugins there is no point of having an old 0.4 client/server since only thing you could achieve would be to join a blank server and do nothing at all
(unless you find a vcmp.h sdk from 2015 and make a gamemode with c++) 
* Note 3: Windows Server executable gets installed to your root directory when you install 0.1x clients
